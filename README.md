Why this repository exists

This repository contains the initial experimental work that laid the foundation for what later became the Qazna open financial standard.

At the time of its creation, the project was developed as a startup prototype, exploring architectural, technical, and conceptual directions for a new type of financial infrastructure.

Over time, the project outgrew the startup framing and evolved into something more fundamental:

a vendor-neutral, open financial standard, comparable in role to how Linux functions in operating systems.
